# Boothera
Final project
Boothera - Event Booth Rental Platform

Project Overview
Boothera is a streamlined online platform designed to connect event organizers with vendors. It simplifies the booth rental process, allowing users to browse available spaces, submit applications, and manage their bookings through a centralized, user-friendly dashboard.
----------------------------------------------------------------------
Key Features:
Booth Discovery: Advanced filtering and search for upcoming event spaces.
Application Management: Track real-time application status via the user dashboard.
User Profile: Centralized management of vendor information and rental history.
Branding: Features a clean, mountain-themed visual identity.
-----------------------------------------------------------------------
Setup Instructions
1. PrerequisitesNode.js: v18.0 or higherPackage Manager: npm or yarnDatabase: Ensure your local or development database is running.

2. Installation StepsBash
# Clone the repository
git clone <https://github.com/crayonshinchan198555-dot/Boothera.git>

# Navigate to the project directory
cd boothera

# Install dependencies
npm install

# Configure environment variables
# Copy the example file and update with your API keys and DB URI
cp .env.example .env

# Start the development server
npm run dev
